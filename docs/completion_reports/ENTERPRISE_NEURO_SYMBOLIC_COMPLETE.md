## 🎯 Quick Start

### Enterprise Neuro-Symbolic AI
```python
from ml_intelligence.enterprise_neuro_symbolic import get_enterprise_neuro_symbolic
from ml_intelligence.neuro_symbolic_reasoner import get_neuro_symbolic_reasoner

# Get reasoner
reasoner = get_neuro_symbolic_reasoner(
    retriever=retriever,
    learning_memory=learning_memory,
    neural_weight=0.5,
    symbolic_weight=0.5
)

# Get enterprise wrapper
enterprise_neuro_symbolic = get_enterprise_neuro_symbolic(
    neuro_symbolic_reasoner=reasoner,
    retention_days=90
)

# Perform reasoning (with tracking)
result = reasoner.reason(query="authentication system")
enterprise_neuro_symbolic.track_reasoning(
    result=result,
    reasoning_time_ms=2500.0,
    phase_times={
        "neural_search": 800,
        "symbolic_query": 600,
        "cross_inform": 400,
        "fusion": 700
    }
)

# Get analytics
analytics = enterprise_neuro_symbolic.get_neuro_symbolic_analytics()

# Optimize weights (automatic)
optimization = enterprise_neuro_symbolic.optimize_weights()
```

## 🎯 Weight Optimization

The enterprise system **automatically optimizes** neural-symbolic weights based on performance:

```python
# System analyzes last 100 reasonings
optimization = enterprise_neuro_symbolic.optimize_weights()

# Returns:
{
    "optimized": True,
    "optimal_neural_weight": 0.6,  # Adjusted based on performance
    "optimal_symbolic_weight": 0.4,
    "current_neural_weight": 0.5,
    "current_symbolic_weight": 0.5,
    "neural_success_ratio": 0.65,
    "symbolic_success_ratio": 0.35
}
```

**How It Works**:
1. Analyzes last 100 reasonings
2. Calculates success rates for neural vs symbolic
3. Adjusts weights proportionally
4. Only adjusts if difference > 0.1 (to avoid thrashing)
