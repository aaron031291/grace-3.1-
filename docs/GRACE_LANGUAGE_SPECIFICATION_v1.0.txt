================================================
FORMULATION RULES
================================================

1. BASE STRUCTURE
   - 6-dot matrix (Braille standard)
   - 3 spacing levels: 1mm (tight), 2mm (medium), 3mm (wide)
   - 4 shape modifiers: □ (square), △ (triangle), ○ (circle), □△○ (combined)

2. MEMORY CONTEXT (Spacing)
   - 1mm = Procedural (how you do it)
   - 2mm = Episodic (when you did it)
   - 3mm = Semantic (what it means)
   - +Shape = Ghost (temporary context)

3. LOOP ASSIGNMENT
   - HOMEOSTASIS: Fix, Broken, Not working, Debug, Errors, Problems, Healing
   - COGNITION: Check, Identify, Find, Map, Trace, Decide, Think
   - LEARNING: Learning, Knowledge, Educate, Memory, Train, Adapt
   - GOVERNANCE: Verify, Validate, Trust, Score, Pillar, Audit
