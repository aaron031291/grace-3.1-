# feedback_loops

Grace Feedback Loops (5 core loops, consolidated from 11)

## Location
`cognitive/feedback_loops.py`

## Classes
- **QualityFeedback**: Loop 1: Merged criteria + research + verification guard feedback.
- **KnowledgeGaps**: Loop 2: Merged gap priority + question effectiveness.
- **PlaybookEvolution**: Loop 3: Playbooks improve with each use.
- **WeightBackprop**: Loop 4: Merged weight propagation + cross-task dependencies.
- **TaskEnrichment**: Loop 5: Enriches tasks with unified intelligence context.
- **FeedbackCoordinator**: Single coordinator for all 5 loops.
