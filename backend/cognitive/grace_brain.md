# grace_brain

Grace Brain - READ-ONLY Intelligence Layer

## Location
`cognitive/grace_brain.py`

## Classes
- **InstructionType**: Types of instructions Kimi can produce.
- **InstructionPriority**: Priority of an instruction.
- **GraceDiagnosis**: Grace's diagnosis of the system state.
- **KimiInstruction**: A single instruction Kimi produces for Grace to execute.
- **GraceInstructionSet**: A set of instructions Kimi produces from a single analysis session.
- **GraceBrain**: Grace's read-only intelligence layer.
