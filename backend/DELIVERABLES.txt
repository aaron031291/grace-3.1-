================================================================================
FEATURES OF reset_and_reingest.py
================================================================================

WHAT IT DOES:
  1. Clears all SQLite database data
     - Documents
     - Document chunks
     - Chats
     - Chat histories

  2. Clears all Qdrant vector database data
     - Deletes documents collection
     - Removes all embeddings

  3. Resets file ingestion tracking
     - Clears .file_states.json
     - Resets git tracking

  4. Triggers fresh auto-ingestion
     - Scans knowledge_base directory
     - Re-ingests all files
     - Reports progress and results

LOGGING FEATURES:
  ✓ Color-coded output (green/yellow/red)
  ✓ Timestamps with millisecond precision
  ✓ Progress indicators (1/4, 2/4, 3/4, 4/4)
  ✓ File list before ingestion
  ✓ Per-file ingestion status
  ✓ Processing time per file
  ✓ Summary statistics
  ✓ Total duration
  ✓ Clear visual separators
  ✓ Comprehensive error messages
