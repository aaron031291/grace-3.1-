## probe_agent_api.py

Discovers all registered routes from FastAPI's app.routes\n  2. Sends a lightweight synthetic request to each endpoint\n  3. Tags every probe with a Genesis key (test=true so metrics aren't polluted)\n  4. Classifies results: ACTIVE (2xx), DORMANT (no recent Genesis keys),\n     BROKEN (4xx/5xx/timeout)\n  5. Broken endpoints trigger self-healing via consensus\n\nThe consensus auto-fix flow:\n  1. Probe finds a broken endpoint\n  2. Sends the error to ALL available models (Kimi, Opus, Qwen, DeepSeek)\n  3. Each model proposes a diagnosis + fix\n  4. If all models agree в†’ execute the fix (with Genesis key provenance)\n  5. If disagreement в†’ queue for human approval\n"
