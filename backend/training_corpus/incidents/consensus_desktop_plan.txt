2. REVERSE-kNN: FIND WHAT THE ORACLE **DOESN’T** KNOW
   (code you can drop into `scripts/gaps.py`)

   ```python
   import numpy as np, qdrant_client, pandas as pd
   from qdrant_client.http import models as q

   client = qdrant_client.QdrantClient(host="localhost", port=6333)
   COLLECTION = "grace_kb"

   # 1. Pull every embedding once
   ids, vectors = [], []
   for offset in range(0, 1_000_000, 10_000):
       r = client.scroll(COLLECTION, offset=offset, limit=10_000, with_payload=False, with_vectors=True)
       if not r: break
       ids   += [p.id for p in r[0]]
       vectors += [p.vector for p in r[0]]
   X = np.array(vectors, dtype=np.float32)

   # 2. Build an *exact* kNN index (Faiss CPU is fine for <1 M)
   import faiss
   index = faiss.IndexFlatIP(X.shape[1])   # inner-product = cosine if unit vectors
   faiss.normalize_L2(X)
   index.add(X)

   # 3. Reverse sweep: for each point, how many neighbours within radius τ ?
   τ = 0.75   # cosine <0.75 => “far”
   low_coverage = []
   for i, v in enumerate(X):
       D, I = index.search(v.reshape(1, -1), k=200)
       near = (D >= τ).sum()
       if near < 5:                      # <5 close neighbours → gap
           low_coverage.append((ids[i], near, 1-D[0][1]))   # id, count, dist-to-NN

   # 4. Cluster gaps by metadata (here by folder tags)
   df = pd.DataFrame(low_coverage, columns=["id","neighbours","nn_dist"])
   payload = client.retrieve(COLLECTION, ids=df.id.tolist(), with_payload=["path","title"])
   df["folder"] = df.id.map({p.id: p.payload["path"].split("/")[0] for p in payload})
   gaps = df.groupby("folder").agg({"neighbours":"mean","nn_dist":"mean","id":"count"})
   print("Knowledge gaps (top folders):")
   print(gaps.sort_values("neighbours").head(10))
   ```
   Run nightly; anything with <3 neighbours or >0.8 NN-distance lands on the “write-more-here” list.

4. GUARDED COMPILER – HOW TO SHIP CODE SAFELY TODAY
   Grace now auto-sandboxes any generated source file:

   - Python → `RestrictedPython` compile + `resource.setrlimit(RLIMIT_CPU, 10 s)`
   - C/C++ → compile with `-O0 -fsanitize=address,undefined` inside `podman` with `--memory=1g --cpus=0.5 --no-new-privileges`
   - Anything else → `firejail --seccomp --private-tmp --quiet`

   The runtime verifier blocks the PR until tests pass; no manual step needed.
