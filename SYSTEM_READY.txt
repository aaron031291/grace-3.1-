HOW TO USE
================================================================================

STEP 1: Ensure Qdrant is Running
  $ docker start qdrant
  $ curl http://localhost:6333/health
  Expected: {"status": "ok"}

STEP 2: Upload a File
  Web UI:
    1. Open http://localhost:3000
    2. Go to Documents → Files tab
    3. Click upload and select a file
    
  API:
    $ curl -X POST http://localhost:8000/files/upload \
      -F "file=@yourfile.pdf"

STEP 3: Search
  Web UI:
    1. Go to Documents → Search tab
    2. Type your query
    3. Press Enter
    
  API:
    $ curl -X POST "http://localhost:8000/retrieve/search?\
      query=your+search&limit=5&threshold=0.3"
