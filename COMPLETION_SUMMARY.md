## 🚀 How to Use

### Quick Start (5 minutes)

```bash
# Terminal 1: Ollama
ollama serve

# Terminal 2: Backend
cd backend && python app.py

# Terminal 3: Frontend
cd frontend && npm run dev

# Open browser
open http://localhost:5174
```

### First Chat

1. Click the **+** button to create a new chat
2. Type your message in the input field
3. Press Enter or click send
4. Wait for AI response
5. Continue the conversation

### Manage Chats

- **Rename**: Click the ✎ button on a selected chat
- **Delete**: Click the 🗑 button to delete (with confirmation)
- **Create**: Click the **+** button in the header
- **Switch**: Click any chat in the list

## 🔄 Integration Points

### APIs Used (8 endpoints)

1. `GET /health` - Health check
2. `POST /chats` - Create chat
3. `GET /chats` - List chats
4. `GET /chats/{id}` - Get chat
5. `PUT /chats/{id}` - Update chat
6. `DELETE /chats/{id}` - Delete chat
7. `GET /chats/{id}/messages` - Get history
8. `POST /chats/{id}/prompt` - Send message

### Data Flow

- Frontend sends HTTP requests to backend
- Backend processes and calls Ollama
- Responses stored in database
- Frontend displays messages in real-time
